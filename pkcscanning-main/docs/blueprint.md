# **App Name**: PlateSeeker

## Core Features:

- Live OCR Scanning: Real-time optical character recognition of car number plates using the device's camera.
- Plate Number Display: Clearly display the detected number plate for easy viewing and confirmation.
- Plate Saving: Allow users to save the detected plate numbers for later reference.
- Scan History Log: Maintain a history log of all scanned plate numbers, with timestamps.

## Style Guidelines:

- Primary color: Dark slate blue (#37718e), evoking reliability
- Background color: Near-black (#05080a) evoking camera darkness and focusing the user on the scan target, which is crucial
- Accent color: Pale turquoise (#90dbf4), contrasting with the dark primary for emphasis on the display plate
- Body and headline font: 'Inter', a sans-serif for a modern, machined feel.
- Use clear, minimalist icons for scan, save, and history functions.
- Camera view takes up the majority of the screen with controls overlaid.
- Subtle animation to highlight detected plates, confirming successful scans.
